# Library-Management-System

# Description
This is a Library Management System that I built using C++

# Features
Adding books and magazines
Removing books and magazines
Search books and magazines
Display books and magazines

# How to Run
Have a C++ compiler (g++) installed
Compile the program:
g++ main.cpp -o library
Run the program:
./library
